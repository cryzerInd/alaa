# موقع علاء عبد الكريم — رسام معماري ومدخل بيانات على منصة بلدي

موقع شخصي احترافي يعرض السيرة الذاتية والخدمات والمهارات والخبرات.

## 📁 ملفات الموقع
```
Alaa/
├── index.html      ← الصفحة الرئيسية (كل المحتوى)
├── css/style.css   ← التصميم والألوان والحركات
├── js/script.js    ← التفاعل (القائمة، الحركة، العدادات)
└── assets/         ← مجلد للصور (ضع صورتك الشخصية هنا)
```

## 👀 معاينة الموقع على جهازك
افتح ملف `index.html` بالضغط عليه مرتين — سيفتح في المتصفح مباشرة.

## 🌐 النشر مجاناً على الإنترنت (اختر طريقة واحدة)

### الطريقة 1 — Netlify Drop (الأسهل، بدون حساب تقني)
1. ادخل على: https://app.netlify.com/drop
2. اسحب مجلد `Alaa` كاملاً وأفلته في الصفحة.
3. خلال ثوانٍ سيظهر رابط موقعك مجاناً (مثل: `your-site.netlify.app`).
4. تقدر تغيّر الاسم من إعدادات الموقع، وتربطه بدومين خاص لاحقاً.

### الطريقة 2 — GitHub Pages (الأفضل للاستمرارية)
1. أنشئ حساب على https://github.com
2. أنشئ مستودع (Repository) جديد باسم: `Alaa`
3. ارفع كل الملفات داخله.
4. من **Settings → Pages** اختر الفرع `main` والمجلد `/ (root)` ثم Save.
5. رابط موقعك الحالي: `https://cryzerind.github.io/alaa/`

### الطريقة 3 — Vercel
1. ادخل https://vercel.com وسجّل بحساب GitHub.
2. اختر Import للمستودع ثم Deploy.
